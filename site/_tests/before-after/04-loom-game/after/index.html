Loom

Weave threads of light through a world of shadows. Every path is a choice. Every choice reshapes the dark.

The game

Loom is a meditative puzzle game about connection. You are the weaver — drawing threads of light across a dark lattice, finding the pattern that holds. Each of its 120 hand-built levels is a small world. Each solution, a small illumination.

01

Light follows intention.

Every thread you place remembers where it came from. Pull a strand through a dark node and it carries the glow forward — illuminating what lies beyond only if the path was deliberate. There is no undo. There is only the weave.

02

120 handcrafted levels.

No procedural filler. Each puzzle built by hand, tested until it had exactly one elegant solution.

03

Made to be quiet.

No timer. No score. Ambient sound only. Loom is a game you sit inside.

04

A world that responds.

The lattice shifts as your threads settle. What you illuminate changes what comes next.

05

No tutorials.

The first thread teaches you everything. The rest is discovery.
