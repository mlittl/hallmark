Designs that refuse to look AI-generated.

Hallmark is a skill for Claude Code, Cursor, and Codex. It encodes the anti-slop consensus — typography, colour, layout, motion, interaction — into one holistic ruleset your AI assistant will actually follow.

02⁄The problem

Every LLM was trained on the same templates. Without a firm hand, they all emit the same page.

Without Hallmark

Centred hero. Purple-to-pink gradient. Inter display. Startup-cliché tagline. Rounded-pill CTA. A view every LLM has produced ten thousand times.

With Hallmark
01 ⁄ Specimen A quiet instrument.

Asymmetric, left-biased. One accent, barely visible. A distinctive display face paired with a serif for emphasis. No gradient, no pill, no slop.

03⁄Foundations

Eight dimensions. A short, opinionated rule on each.

Typography F / 01

Banned defaults. Distinctive display paired with a refined body. Commit to weight extremes.

Display face paired with a workhorse body.
Colour F / 02

OKLCH only. One accent, under 3% of any view. No pure black. No pure white. No purple gradients.

Space F / 03

4-point scale, named semantically. gap for siblings. Break the grid on purpose, not by accident.

4 ⁄ 8 ⁄ 12 ⁄ 16 ⁄ 24 ⁄ 40
Motion F / 04

Animate transform and opacity only. One orchestrated entrance beats ten micro-bounces.

cubic-bezier(0.16, 1, 0.3, 1)  ·  420 ms
States F / 05

Eight states, not two. :focus-visible ring. 44 px hit target. Labels above inputs.

default hover focus active disabled loading error success
Responsive F / 06

Mobile-first. Breakpoints where content breaks, not where a device sits. dvh, not vh.

40 rem   60 rem   90 rem
Copy F / 07

Specific verbs. Three-part errors. Link text that reads alone. Proper punctuation, not straight quotes.

“Save changes” beats “OK”. “View pricing plans” beats “Click here”.

05⁄How it works

Install once. Describe what you need. Hallmark routes the rest.

I

Install the skill

One command. Works in Claude Code, Cursor, and Codex out of the box.

$npx skills add hallmark
II

Ask for a UI

In plain language. Hallmark refuses to design without a confirmed audience, use case, and tone — so it asks first.

> build a landing page for a local bookshop's
  new poetry collection. editorial tone,
  for literary-minded customers.
III

Ship, audit, refine

The default verb builds. hallmark audit scores existing code against the tells. hallmark refine polishes in place.

$ claude "hallmark audit ./components/hero.tsx"
  → 2 critical · 3 major · 1 minor
  → verdict: reads as AI-generated
